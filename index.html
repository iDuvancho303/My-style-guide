Text Styles

H1: Main page heading

  • Should be used one time only
  • Font-weight: 700 (bold)
  • Font-size: 33px
  • Font-family: Nunito Sans
  • Color: #FFFF33

H2: Subheading

  • You can use any number of H2-H6 elements on any page
  • Font-weight: 500
  • Font-size: 18px
  • Font-family: Poppins
  • Color: #0000FF

P: Paragraph text

  • Can be used as many times as you want
  • Font-weight: 400 (regular)
  • Font-size: 14px
  • Font-family: Poppins
  • Color: #FFFF00
